# Project Showcase

This repository contains a collection of standalone web applications built with Vue and PHP. For now it has only one application but there will be more soon.

## Applications

### 1. Bank Site

A well secured banking site that allows users to create credit cards, depit cards, deposit and transfer.

- **Features:** Authentication, Authorization, JWT tokens, Envelope Encryption, Fingerprint use, Data Encryption, Rate Limiting.
- **Technologies:** Vue, JavaScript, PHP, MySQL, Tailwind CSS, HashiCorp Vault

## Getting Started

Each application has its own set of dependencies and setup instructions. Please refer to the `README.md` file within each application's directory for detailed instructions on how to run them locally.
